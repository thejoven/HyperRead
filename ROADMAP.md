# 🗺️ HyperRead 发展路线图

## 📖 项目愿景

HyperRead 致力于成为最优秀的 Markdown 阅读器，专注于提供卓越的文档阅读体验。我们的目标是打造一个既美观又实用的工具，让用户能够更加专注、高效地阅读和理解 Markdown 文档。

---

## ✅ 已完成功能

### v1.2.0 - 多语言与智能刷新
- **🌍 完整多语言支持** - 支持中英文界面切换，适应全球用户需求
- **🔄 智能文件刷新系统** - 解决拖拽模式下文件重命名后消失的关键问题
- **📚 示例文档系统** - 基于认知科学的 Markdown 学习教程
- **🖼️ 本地图片显示支持** - 完整支持 Markdown 中的本地图片，智能路径解析和多格式支持
- **🔍 图片放大预览功能** - 类似 Mermaid 图表的全屏预览模式，支持缩放、拖拽和位置控制
- **🛠️ TypeScript 类型安全** - 增强代码质量和开发体验

### v1.1.0 - Mermaid 图表增强
- **🎯 全屏图表查看** - 点击图表右上角按钮进入沉浸式查看模式
- **🔍 缩放控制系统** - 支持 0.1x - 5.0x 精确缩放控制
- **🖱️ 拖拽手势控制** - 启用拖拽模式后可自由移动图表查看不同区域
- **🔗 外部链接支持** - Markdown 中的外部链接自动在系统默认浏览器中打开

### v1.0.0 - 核心功能
- **🎨 原生 macOS 界面** - 完美融入 macOS 设计语言
- **📁 拖拽支持** - 支持单文件和文件夹拖拽加载
- **🌳 文件树浏览** - 递归扫描并展示目录结构
- **⚡ 纯本地运行** - 无需开发服务器，启动更快
- **🎯 完整 Markdown 渲染** - 支持所有标准语法和扩展功能

## 📅 近期计划 (Q1 2025)

### 🏆 阅读体验核心优化

#### 📖 沉浸式阅读模式
- **阅读进度指示** - 显示当前文档的阅读进度和预计剩余时间
- **自动滚动功能** - 可调节速度的自动滚动，适合长文档阅读

#### 🎨 视觉舒适度提升
- **动态行高调整** - 基于内容密度智能调整行高
- **色彩对比度优化** - 提供多种预设配色方案，确保最佳可读性
- **夜间模式增强** - 更柔和的深色主题，减少眼部疲劳

#### 📐 布局与排版优化
- **响应式布局** - 支持不同窗口尺寸下的最佳阅读体验
- **段落间距智能调整** - 根据内容类型自动调整段落间距
- **图表与文本协调** - 优化图表在文档中的展示和布局
- **代码块美化** - 更好的代码高亮和格式化显示

### 📚 文档管理增强

#### 🗂️ 历史记录系统
- **阅读历史追踪** - 记录最近打开的文档和阅读位置
- **书签功能** - 为重要段落添加书签，快速跳转
- **阅读时长统计** - 追踪每个文档的阅读时间
- **搜索历史** - 保存搜索关键词，提供智能建议

#### 🔍 高级搜索功能
- **全文搜索** - 在当前文档或整个文件夹中搜索内容
- **正则表达式支持** - 支持复杂的搜索模式
- **搜索结果高亮** - 在文档中高亮显示搜索结果
- **搜索导航** - 快速在搜索结果间跳转

### ⚡ 性能与响应速度

#### 🚀 加载性能优化
- **增量渲染** - 大文档分段渲染，提升加载速度
- **内存管理优化** - 更高效的内存使用，支持更大文档
- **缓存策略** - 智能缓存已渲染内容，加快重复访问速度
- **懒加载图片** - 图片按需加载，减少初始加载时间

## 🎯 核心设计原则

### 🔍 以阅读为中心
- 所有功能设计都围绕提升阅读体验展开
- 避免不必要的功能膨胀，保持简洁专注
- 优先考虑内容的可读性和视觉舒适度

### 🎨 原生体验
- 深度集成操作系统特性，提供原生应用体验
- 遵循平台设计规范，确保界面一致性
- 充分利用硬件特性，优化性能表现

### ⚡ 性能优先
- 快速启动，即开即用
- 流畅的交互响应，无卡顿体验
- 高效的内存和 CPU 和 GPU 使用

### 🔧 可定制性
- 丰富的个性化设置选项
- 灵活的界面布局调整
- 可扩展的插件系统（未来）
